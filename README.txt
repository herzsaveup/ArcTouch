* Special Build Instructions:

- Install Cocoapods (Here's a guide on how to install it just in case -> https://www.raywenderlich.com/97014)
- Execute 'pod install' in the source code root folder to install Cocoapods dependencies.

* Third-party libraries with Cocoapods:

- SDWebImageCache: UIImageView category used to cache and manage image downloads asynchronously to have smooth scrolling thorugh movies list while downloading movies posters.
- SVProgressHUD: Simple library to show waiting spinner while downloading information to be displayed in the app.
